## 📜 项目简介 (Introduction)

**《赤色纪元：殊途同归》** 是一款宏大的**历史推演视觉小说**，时间跨度长达 140 年（1912 - 2050）。

玩家将扮演不同历史时期的关键决策者——从辛亥革命的热血青年，到建国初期的国家领袖，再到21世纪的赛博变革者。你的每一个选择，不仅决定了国家的命运，更将分裂出三条截然不同的世界线。

本作采用独特的**90年代街机（Arcade / CPS1）美术风格**，致敬《三国志》、《名将》等经典游戏，用像素与高对比度的色彩重构那段波澜壮阔的历史。

## 🌌 三大世界线 (The Three Timelines)

在1976年的关键节点，历史将发生剧烈的分叉：

### 1. 🟥 赤色未来线 (The Red Future)
> **“为了人类的补完，我们选择了机械飞升。”**
*   **核心概念**：赛博共产主义 (Cyber-Communism) / OGAS系统 / 红色美学
*   **描述**：拒绝资本的诱惑，依靠大数据与控制论实现计划经济的极致。2050年的中国建立起了星际舰队与蜂巢思维，实现了另一种形式的“大同”，但也付出了人性的代价。

### 2. 🟦 蓝线 (The Blue Cyberpunk)
> **“欢迎来到夜之城……或者说是南京。”**
*   **核心概念**：财阀统治 (Corporate Sovereignty) / 极端资本主义 / 高科技低生活
*   **描述**：1949年划江而治后的异化推演。泛亚财阀控制了一切，霓虹灯下是无尽的贫民窟。这曾是许多人向往的自由，最终却演变成了最彻底的奴役。

### 3. 🟨 历史线 (The Historical Reality)
> **“摸着石头过河，我们创造了奇迹，也留下了遗憾。”**
*   **核心概念**：改革开放 / 实用主义 / 盛世下的阴影
*   **描述**：我们所熟知的现实世界。经济腾飞，大国崛起，但在物质极度丰富的2050年，人们的精神家园却日渐荒芜。

## 🎮 游戏特色 (Features)

*   **蝴蝶效应系统**：1919年的一个微小选择，可能导致2050年的北京变成一座浮空城。
*   **街机化叙事**：
    *   **UI 设计**：模仿 CRT 显示器的扫描线效果，血条式的属性显示。
    *   **角色立绘**：采用《三国志》风格的选人框与动态像素演出。
    *   **音效**：复古的合成器波形音乐 (Synthwave / Chiptune)。
*   **国家属性管理**：
    *   🛠️ **生产力 (Productivity)**：决定物质基础。
    *   🧠 **阶级觉悟 (Class Consciousness)**：决定社会形态。
    *   ⚖️ **社会稳定 (Stability)**：决定政权存续。
    *   🌍 **国际压力 (Intl Pressure)**：决定外部环境。

## 🚀 快速开始 (How to Play)

1.  克隆本仓库：
    ```bash
    git clone https://github.com/a1050154895/-.git
    ```
2.  下载并安装 [Ren'Py 引擎](https://www.renpy.org/)。
3.  打开 Ren'Py，选择 `RedEra` 文件夹所在的目录。
4.  点击 **"Launch Project"** 开始体验。

## 🛠️ 技术栈 (Tech Stack)

*   **Engine**: Ren'Py 8.0+ (Python 3 based)
*   **Art**: Custom Arcade Style (Generated via Python/PIL & Ren'Py ATL)
*   **Audio**: Procedural Generation (Python `wave` module)
