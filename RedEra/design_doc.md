# 《赤色纪元：殊途同归》 (Red Era: Divergent Paths) - 设计白皮书

## 1. 项目概述
* **类型：** 历史沉浸式视觉小说 (Visual Novel) + 轻度大战略 (Grand Strategy Lite)
* **引擎：** Ren'Py (基于 Python)
* **核心理念：** 唯物史观推演。探讨“物质决定意识”与“关键抉择改变历史”的辩证关系。
* **开发模式：** 单人 + AI 辅助开发

## 2. 世界观与时间线
游戏跨度为 1912-2050 年，包含三个核心平行世界：
1.  **历史线 (The History):** 现实世界线，基准参照。
    *   **路径:** 1949 建国 -> 改革开放 -> 2050 现代化强国。
2.  **蓝线 (The Blue Path - 财阀赛博):**
    *   **触发点:** 1948年内战抉择导致“划江而治”或国民党反扑。
    *   **特征:** 蒋介石政权统治至2050。实行国家资本主义，后期演变为财阀治国。贫富差距极极极大，科技发达但底层生活困苦（赛博朋克风格）。
    *   **关键事件:** 接受美国援助、镇压工运、冷战桥头堡、反攻大陆（或长期对峙）、2050 机械化独裁。
3.  **赤色未来线 (The Red Future - 赛博共产):**
    *   **触发点:** 1976年后的不同选择（坚持阶级斗争+科技爆发）。
    *   **特征:** 2050年物质极大丰富，国家机器消亡，AI 辅助计划经济，人类追求纯粹精神。

## 3. 核心玩法循环
*   **每日决策:** 玩家扮演不同时代的“关键人物”或“决策者”，阅读情报/报纸/内参。
*   **资源管理:** 平衡 4 个核心国家属性，属性归零会导致 Game Over。
*   **历史节点:** 触发重大历史事件，选择决定世界线走向。

## 4. 核心数值系统 (Global Variables)
*   **Productivity (生产力):** 决定物质基础。过低 -> 饥荒/动乱；极高 -> 解锁“共产主义”。
*   **Class_Consciousness (阶级觉悟):** 决定社会形态。高 -> 公有制；低 -> 拜金主义。
*   **Stability (社会稳定):** 国家凝聚力。过低 -> 内战/分裂。
*   **Intl_Pressure (国际压力):** 外部封锁程度。过高 -> 战争/经济崩溃。

## 6. 美术设计与视觉指南 (Art Direction)

### 6.1 核心风格：复古街机 (Retro Arcade / CPS1 Style)
响应玩家号召，本作将全面致敬 90 年代经典街机《三国志 (Warriors of Fate)》。
*   **视觉基调:** 像素风 (Pixel Art)、高饱和度、硬朗的黑色描边。
*   **UI 风格:** 
    *   **头像:** 经典的矩形头像框，带有生命值槽 (Health Bar) 风格的数值显示。
    *   **字体:** 带有厚重黑色描边的黄色/渐变色字体。
    *   **对话框:** 蓝底白框，模仿街机过场动画。

### 6.2 三大世界线变体
虽然统一为街机风，但三个世界线有不同配色倾向：
*   **历史线:** 经典街机风 (Classic Arcade) - 像《三国志》原本的画风，土黄、军绿为主。
*   **蓝线:** 赛博街机风 (Cyber Arcade) - 类似《名将 (Captain Commando)》，霓虹色、高对比度。
*   **赤色线:** 苏式街机风 (Soviet Arcade) - 类似《赤色要塞》，红金配色，金属质感。

### 6.3 角色立绘设计 (街机卡牌版)
角色将以“选人画面”的卡牌形式呈现：
*   **背景:** 纯色或简单的像素抖动图案。
*   **文字:** 巨大的、带有阴影的角色名。
*   **边框:** 模拟 CRT 显示器的圆角或金属边框。

# 人物设定集

## 1. 核心主角/导师
### 毛泽东 (Mao Zedong)
*   **定位:** 精神领袖，唯物辩证法化身。
*   **台词风格:** 宏大、哲学、引用经典语录。

## 2. 玩家主要交互对象 (队友/参谋)
### 陈赓 (Chen Geng)
*   **定位:** 军事/特工顾问。
*   **性格:** 幽默、豪爽、人脉广。
*   **功能:** 军事冲突、情报、外交。

### 鲁迅 (Lu Xun)
*   **定位:** 文化/觉悟顾问。
*   **性格:** 犀利、冷峻、悲悯。
*   **功能:** 提升【阶级觉悟】。

### 钱学森 (Qian Xuesen)
*   **定位:** 科技/未来顾问。
*   **形态:** 1950s 科学家 -> 2050s 数字生命/AI。
*   **功能:** 提升【生产力】。

## 3. 反派/对立面
### 蒋介石 (Chiang Kai-shek)
*   **定位:** 蓝线代表。
*   **性格:** 独断、传统、多疑。
